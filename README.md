This repository demonstrates a common error encountered when using React Router v6: unexpected route matching. The `bug.js` file contains a simple React app with three routes.  The issue lies in how the routes are defined, leading to incorrect component rendering or unexpected navigation behavior. The `bugSolution.js` provides a corrected version of the application, showcasing how to avoid this problem.  This example is useful for understanding and troubleshooting route matching problems in React Router v6.
